Verificador de Números Primos Neón 🚀

Un aplicativo web moderno, interactivo y visualmente impactante diseñado para determinar si un número entero es primo o no. Este proyecto combina lógica matemática con una estética "Cyberpunk/Neón" y efectos visuales dinámicos.

✨ Características

Lógica de Verificación Optimizada: Utiliza un algoritmo eficiente que evalúa hasta la raíz cuadrada del número para una respuesta instantánea.

Estética Neón: Interfaz oscura con efectos de brillo, sombras dinámicas y tipografía moderna.

Fondo Animado: Sistema de partículas de números aleatorios que flotan continuamente con colores neón.

Efectos de Celebración: Al detectar un número primo, se dispara una lluvia de confeti de colores.

Efectos de Explosión: Si el número no es primo, se genera una explosión de partículas rojas intensas.

Diseño Responsivo: Adaptable a dispositivos móviles, tablets y computadoras de escritorio.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica del aplicativo.

Tailwind CSS: Para el diseño visual, efectos de desenfoque (backdrop-blur) y utilidades de espaciado.

JavaScript (Vanilla): Lógica de cálculo, manipulación del DOM y sistema de animación de fondo.

Canvas Confetti: Librería externa para los efectos visuales de celebración y explosión.

📁 Estructura del Archivo

El proyecto está contenido íntegramente en un solo archivo (index.html), lo que facilita su portabilidad y ejecución inmediata en cualquier navegador sin necesidad de compilación o servidores complejos.

🚀 Cómo ejecutarlo

Descarga el archivo index.html.

Ábrelo en tu navegador favorito (Chrome, Firefox, Edge, etc.).

Ingresa un número en el campo central y presiona "Verificar Número" o la tecla Enter.
